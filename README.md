# **Bookmark Manager**

### User Stories
```
As a Makers Student
So that I can keep track of my links
I would like to bookmark them on a page

As a Makers Student
So that I can keep up with new websites
I would like to add links to my bookmark page

As a Makers Student
So that I can organize my links
I want to be able to tag them.

As a Makers student
So that I can search for my links faster
I would like to filter my links
```
